Bases: banditpylib.learners.ordinary_learner.utils.OrdinaryLearner

Thompson Sampling policy [AG17]

Assume a prior distribution for every arm. At time \(t\), sample a virtual mean from the posterior distribution for every arm. Play the arm with the maximum sampled virtual mean.

Warning

Reward should be Bernoulli when Beta prior is chosen.

Bases: banditpylib.learners.ordinary_learner.utils.OrdinaryLearner

UCBV policy [AMSzepesvari09]

At time \(t\), play arm

\[\mathrm{argmax}_{i \in [0, N-1]} \left\{ \hat{\mu}_i(t) + \sqrt{ \frac{ 2 \hat{V}_i(t) \ln(t) }{T_i(t)} }+ \frac{ b \ln(t) }{T_i(t)} \right\}\]

Note

Reward has to be bounded within \([0, b]\).

class banditpylib.learners.ordinary_learner.MOSS(arm_num: int, horizon: int, name=None)[source]¶

Bases: banditpylib.learners.ordinary_learner.utils.OrdinaryLearner

MOSS policy [AB09]

At time \(t\), play arm

\[\mathrm{argmax}_{i \in [0, N-1]} \left\{ \hat{\mu}_i(t) + \sqrt{ \frac{\mathrm{max}(\ln( \frac{T}{N T_i(t)} ), 0 ) }{T_i(t)} } \right\}\]

Note

MOSS uses time horizon in its confidence interval. Reward has to be bounded in [0, 1].
