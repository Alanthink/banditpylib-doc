OrdinaryBandit¶

class banditpylib.bandits.OrdinaryBandit(arms: List[banditpylib.arms.utils.Arm])[source]¶

Bases: banditpylib.bandits.utils.Bandit

Ordinary bandit

Arms are indexed from 0 by default. Each pull of arm \(i\) will generate an i.i.d. reward from distribution \(\mathcal{D}_i\), which is unknown beforehand.
