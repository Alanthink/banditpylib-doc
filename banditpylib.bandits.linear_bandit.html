class banditpylib.bandits.linear_bandit.LinearBandit(features: List[numpy.ndarray], theta: numpy.ndarray, var: float = 1.0)[source]¶

Bases: banditpylib.bandits.utils.Bandit

Finite-armed linear bandit

Arms are indexed from 0 by default. Each pull of arm \(i\) will generate an i.i.d. reward from distribution \(\langle \theta, v_i \rangle + \epsilon\), where \(v_i\) is the feature of arm \(i\), \(\theta\) is an unknown parameter and \(\epsilon\) is a zero-mean noise.
