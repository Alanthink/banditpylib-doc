MOSS policy [AB09]

At time \(t\), play arm

\[\mathrm{argmax}_{i \in [0, N-1]} \left\{ \hat{\mu}_i(t) + \sqrt{ \frac{\mathrm{max}(\ln( \frac{T}{N T_i(t)} ), 0 ) }{T_i(t)} } \right\}\]

Note

MOSS uses time horizon in its confidence interval. Reward has to be bounded in [0, 1].
